# Project Laravel UAS PROMNET

## Deskripsi
Project ini merupakan aplikasi berbasis Laravel yang dibuat untuk memenuhi tugas UAS mata kuliah Pemrograman Internet.

## Instalasi
1. Clone repository
2. Jalankan composer install
3. Salin file .env.example menjadi .env
4. Jalankan php artisan key:generate
5. Jalankan php artisan serve

## Versi Laravel
Laravel 10
